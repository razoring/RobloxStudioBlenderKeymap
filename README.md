# RobloxStudioBlenderKeymap
- Overrides Blender (4.0.0) navigation to use "walk" navigation by default.
- Enables Select-Box by default and switches to Tweak when selection is made.
- Swaps Scale-Cage as default scale method.
- Remaps F to center selection.
- No additional imports needed.

## Installation:
Drag-and-Drop the script into ```~\AppData\Roaming\Blender Foundation\Blender\[version]\scripts\addons``` or click "Install from Disk..." under the chevron in Preferences. 

## Keybinds:
### Navigation:
- W: Move relative forward
- A: Move relative left
- S: Move relative back
- D: Move relative right
- Q: Move down
- E: Move up
- Shift: Changes walk speed to slow.
- Scroll-Wheel: Zoom In/Out  
- Right-Mouse Button: Adjust camera angle
### Miscellaneous:
- F: Center selection
- Left-Mouse Button: Selection-Box / Tweak
